# Pithy App — Roadmap: система прогресса и XP

## Концепция

Приложение построено вокруг фраз (например «I'm trying to please both»).
Каждая фраза — это **модуль** из нескольких уроков:

```
Глобальная карта (все фразы-модули)
  └── Модуль «I'm trying to please both»
        ├── Вводный урок       → +1 уровень, +1 unlock-очко
        ├── Тренажёр "trying"  → +XP, +1 уровень, +1 очко
        ├── Тренажёр "please"  → +XP, +1 уровень, +1 очко
        ├── Тренажёр "both"    → +XP, +1 уровень, +1 очко
        └── Босс-урок          → большой XP, модуль завершён → глобальная карта
```

**XP** накапливается внутри модуля (word_choice + phrase_assembly дают по 10 XP за верный ответ).
Когда XP достигает порога — прогресс-бар босса заполняется и замок снимается.
Все XP вкладываются в замок босса и обнуляются — остаются только **уровни** (глобальные).

## 🔄 Фаза 1 — Регистрация (В РАЗРАБОТКЕ)

Первый урок проходится без аккаунта. После завершения — появляется экран регистрации.
Прогресс из localStorage переносится в Supabase без повторного прохождения.

## Ключевые правила XP

| Правило | Детали |
|---|---|
| Источники XP | `word_choice` + `phrase_assembly` (верный ответ) |
| Значение по умолчанию | 10 XP за ответ (настраивается в модуле) |
| Защита от фарма | UNIQUE(user_id, lesson_id, node_id) — только первое прохождение |
| 3-я попытка в сборке фразы | XP начисляется (ответ подсказан, но засчитывается) |
| Повтор урока | Разрешён (для практики), нода показывается как «пройдено», XP не начисляется повторно |
| XP при удалении ноды учителем | Сохраняется у пользователя |
| Уровни | Глобальные, +1 за каждый завершённый урок |
| XP vs уровни | XP тратится на разблокировку босса и уходит; уровни остаются навсегда |
| Офлайн | Уроки только онлайн |
